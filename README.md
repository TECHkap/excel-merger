# excel-merger
Two Excels with unsorted similar content and you want to have it all sorted in one Excel

### Example

##### EXCEL 1: 
*Name*			*Birthday*

BRICE			  1988.09.23

MARIE			  1959.04.09

BRENDA		  2005.12.28

##### EXCEL 2:
*Birthday*		  *City*

2005.12.28    Munich

1988.09.23	  Shangai

1959.04.09	  Dakar


##### GOAL --> EXCEL 3:

*Name*			*Birthday*		*City*
  
BRICE 			1988.09.23  	Shangai

MARIE	  		1959.04.09  	Dakar

BRENDA  		2005.12.28  	Munich
